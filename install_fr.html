Installer

Obtenir toutes les sources

wget https://github.com/Open-UDC/open-udc/archive/v0.2.0.tar.gz

Installer Lud: OpenUDC Client

Obtenir les sources

git clone https://github.com/Open-UDC/lud

Installer

cd lud
make install

Now you can launch lud main menu:

lud.sh

Now you can choose in the menu :

  • It asks you informations (place of birth, date of birth, last name, first name) in order to generate a UDID2 certificate needed to be with you unique OpenGPG/OpenUDC key ID.
  • When your certificate is generated, it shows a menu.
  • Launching the programm "seahorse" you can check your unique UDID2-OpenUDC key is on the window "My personnal keys".
  • Share your KEY with an OpenUDC community Node

Installer OpenUDC Node (ludd)

Préparez les dépendances

apt-get install libgpgme11-dev

Obtenir les sources

git clone https://github.com/Open-UDC/thttpgpd

Installer

cd thttpgpd
./configure --enable-OpenUDC
make
make install

Vérifier si le répertoire /usr/local/var/ludd est crée

ls -a /usr/local/var/ludd

Tapez la commande:

ludd_init.sh
ludd

Testez sur votre explorateur l'adresse : http://127.0.0.1:11371/ et vous accédez au noeud OpenUDC

Vous pouvez réessayer l'installation:

sudo make distclean 
./configure && make 
sudo make uninstall
sudo make install
